Welcome to django-magazine's documentation!
===========================================

django-magazine is a pluggable Django app for managing a simple
magazine.

Magazines consist of Issues, each of which contains one or more
Articles, which are by one or more Authors.


Get the code
-------------

The `source <http://github.com/dominicrodger/django-magazine>`_ is available on Github.
